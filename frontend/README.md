# PPDB Frontend

Frontend PPDB Online menggunakan React + Vite + Tailwind CSS v3.

## Cara Menjalankan
```bash
pnpm install
pnpm dev
```

## Struktur Folder
```
src/
├── components/   → komponen reusable
├── pages/        → halaman aplikasi
├── routes/       → routing & route guard
├── store/        → Zustand global state
├── hooks/        → custom hooks
├── services/     → komunikasi API
└── utils/        → helper functions
```
